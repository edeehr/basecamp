Packer Templates: 
===========

### Basics
- Collection of simple packer templates
- Design objective is to keep this simple as it will be used for demos, basecamps and training purposes

### Directories
----------------------
| Directory | Purpose |
|:-------- |:--------|
1-simple-ami | create linux ami with http and default web page
2-ami-with-php | create linux ami with http, php and customer web page
3-docker | parent director for docker images

### Usage
-----
```
packer <validate|build> <template>
```
